> Gitee


```
git clone https://gitee.com/youzicha123/fluxion.git

LANG=en_US.UTF-8 sudo ./fluxion/fluxion.sh 

# 查看显示网络接口信息
ip link
# 开启网卡
ip link set waln0 up
# 关闭网卡
ip link set waln0 down
# 开启网卡的混合模式
ip link set waln0 promisc on
# 关闭网卡的混合模式
ip link set waln0 promisc off
# 
ip link set waln0 txqueuelen off


```
