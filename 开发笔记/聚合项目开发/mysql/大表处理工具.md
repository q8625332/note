# pt-online-schema-change
>  数据库使用的是行锁，但是修改大表难以避免的是会有锁表的情况。这个工具可以最大程度降低，修改大表带来的影响。
>  这个工具可以加索引，或者增加字段。

[按照教程可以看这个文章](https://blog.csdn.net/qq_45534118/article/details/110365818 "按照教程可以看这个文章")

## 例子：

```
sh  pt-schema-define_schema_table_hytto_threshold_async.sh hytto w_表 " ADD INDEX idx_accountId(accountId) USING BTREE,ADD INDEX idx_serId(userId) USING BTREE "
```
