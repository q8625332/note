> linux 查看文件相关命令

`less` 是一个分页查看文件内容的工具。常用命令：

**基本用法**


```
less filename        # 查看文件

cat file | less      # 管道输入
```


**导航**

- `空格` / `f` - 下一页
- `b` - 上一页
- `g` - 跳到开头
- `G` - 跳到结尾
- `数字g` - 跳到指定行（如 `100g`）

**搜索**

- `/pattern` - 向下搜索
- `?pattern` - 向上搜索
- `n` - 下一个匹配
- `N` - 上一个匹配

**其他**

- `q` - 退出
- `-N` - 显示行号（`less -N file`）
- `-S` - 不换行显示长行
- `F` - 实时跟踪文件（类似 `tail -f`）
